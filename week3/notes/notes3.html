Functions

Declarations

    a function literal looks like this:

    function hello() { console.log("hello world!"); };

    A function can also be defined with a function expression. Like this:

  • const goodbye = function() { console.log("Goodbye"); }; This is an unnamed function.

  • const goodbye = function bye(){ console.log('Goodbye World!'); }; It can also be named like this.

Parameters and arguments

Examples:

function square(x){ return x*x; }

When we invoke this function, we need to provide an argument, which is the number to be squared: square(4.5);

Default Parameters

I found this to be very interesting. This was new to me.

These are values that will be used by the function if no arguments are provided when it is invoked. To specify a default parameter, simply assign the default value to it in the function definition: function hello(name='World') { console.log(`Hello ${name}!); }

Example of a function that has a default parameter of 10

function discount(price, amount=10) { return price*(100-amount)/100; }

So if you call it with a parameter of 20 it will discount 10% off of it a return 18. Like this:

discount(20) // standard discount of 10% result is 18.
You can also change the default parameter like this.
discount(15, 20) // discount of 20% the result will be 12

Arrow functions

Arrow functions can be identified by the 'arrow' symbol, => that gives them their name. The parameters come before the arrow and the main body of the function comes after. Arrow functions are always anonymous, so if you want to refer to them, you must assign them to a variable. For example, the square function we wrote earlier can be written like so:
const square = x => x*x;
